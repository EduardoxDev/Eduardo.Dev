# Eduardo Maciel — Portfolio

Personal portfolio for **Eduardo Maciel**, Software Engineer & CTO at [Stackr Hosting](https://www.stackr.lat/).

Built with **Next.js 14**, **TypeScript**, **Tailwind CSS**, **Framer Motion**, and **MDX**.

---

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

---

## Deploy to Vercel

```bash
# Option A: Vercel CLI
npx vercel

# Option B: Push to GitHub → import at vercel.com/new
```

---

## Customization

All portfolio content lives in **`lib/data.ts`** — edit `PERSONAL`, `PROJECTS`, `EXPERIENCE`, `TECH_STACK`, and `PRINCIPLES`.

Blog posts go in `content/blog/` as `.mdx` files:

```mdx
---
title: "Your Post Title"
description: "Short summary"
date: "2024-06-15"
tags: ["go", "distributed-systems"]
---

Your content here...
```
